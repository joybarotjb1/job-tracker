# 📋 JobTrack – Job Application Tracker

A clean, dark-themed web app to track your job applications — built with pure HTML, CSS, and JavaScript.

## 🔗 Live Demo
> Upload to GitHub Pages to get a live link!

---

## 📸 Features

- ✅ Add job applications with company, role, and date
- ✅ Track status: Applied → Interview → Offer / Rejected / Ghosted
- ✅ Pipeline dashboard with live counts
- ✅ Filter by status
- ✅ Add notes to each application
- ✅ Data saved in browser (localStorage)
- ✅ Fully responsive on mobile

---

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure & layout |
| CSS3 | Styling, animations, responsive design |
| JavaScript (Vanilla) | Logic, DOM manipulation, localStorage |
| Google Fonts | DM Serif Display + DM Sans |

---

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/job-tracker.git
   ```
2. Open `index.html` in your browser — no server needed!
